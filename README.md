# Dynamic-Airtable-Connected-Form-Builder
A full-stack application that allows users to log in with Airtable, create custom forms from Airtable fields, and save responses directly back to Airtable.

## Project Structure
Dynamic-Airtable-Connected-Form-Builder/
│
├── frontend/ # React app
│ ├── public/
│ ├── src/
│ ├── package.json
│ └── README.md
│
├── backend/ # Node.js + Express
│ ├── controllers/
│ ├── models/
│ ├── routes/
│ ├── server.js
│ ├── package.json
│ └── .env.example
│
├── README.md # Main project README
└── .gitignore


## Features

- Airtable OAuth login
- Dynamic form creation using Airtable fields
- Conditional logic for form fields
- Save form responses to Airtable
- Fully responsive frontend

## Setup Instructions

### Backend

1. Go to the backend folder:
cd backend

2. Install dependencies:
# npm install
3. Copy .env.example to .env and fill in your Airtable API keys:
cp .env.example .env
4. Start the backend server:
npm start
5.Frontend
Go to the frontend folder:
cd frontend
6.Start the React app:
npm start
